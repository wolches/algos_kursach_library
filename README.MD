# Сборка
Реализовано при использовании системы сборки CMake 4.0.2, компилятор MSVC 17 
# Описание директорий
- `reader` Класс данных читателя, специфичные для него value object'ы, класс репозитория, опирающийся на хэш-таблицу.
- `book` Класс данных книги, value object кода книги, класс репозитория, опирающийся на АВЛ дерево
- `book_issue` Класс данных истории выдачи книг и репозитория, опирающегося на однонаправленный связный список
- `utils` Вспомогательные классы и функции - в т.ч. реализации алгоритма Бойера-Мура и структур данных, согласно заданию на курсовую работу.
- `app` Логика работы приложения 
- - `app/ui` Пользовательский интерфейс и ввод-вывод
- - `app/service` Бизнес-логика приложения
# Доступный функционал
## Работа с читателями:
1. Регистрация нового читателя
2. Редактирование данных читателя
3. Снятие с обслуживания читателя
4. Просмотр всех зарегистрированных читателей
5. Очистка данных о читателях
6. Поиск читателя по номеру читательского билета
7. Полнотекстовый поиск читателя по ФИО/месту работы/году рождения/ID
## Работа с книгами:
8. Добавление новой книги
9. Редактирование сведений о книге
10. Удаление сведений о книге
11. Просмотр всех имеющихся книг
12. Очистка данных о книгах
13. Поиск книги по шифру
14. Полнотекстовый поиск книги по названию/автору/издательсту/году
## Работа с выдачей книг:
15. Регистрация выдачи экземпляра книги читателю
16. Регистрация приема экземпляра книги от читателя
